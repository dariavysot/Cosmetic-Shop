# Cosmetic Shop

## Опис проєкту
Веб-застосунок **"Магазин косметики"**, розроблений як навчальний проєкт з дисципліни SQL.

Проєкт дозволяє вести облік косметики на складі, формувати купівельну корзину та здійснювати покупки з автоматичним списанням товару.

## Основні функції
* Перегляд залишків косметики на складі
* Формування купівельної корзини покупця (товар, ціна, сума покупки)
* Редагування купівельної корзини
* Автоматичне списання кількості товару при покупці
* Додавання косметики на склад
* Пошук косметики за назвою та постачальником
* Дружній інтерфейс

## Запуск проєкту

1.  Клонувати репозиторій:

    ```bash
    git clone [https://github.com/dariavysot/Cosmetic-Shop.git](https://github.com/dariavysot/Cosmetic-Shop.git)
    cd Cosmetic-Shop/src
    ```

2.  Встановити залежності Laravel:

    ```bash
    composer install
    ```

3.  Налаштувати `.env` (скопіювати `.env.example` і відредагувати під локальну базу даних):

    ```bash
    cp .env.example .env
    php artisan key:generate
    ```

4.  Запустити локальний сервер:

    ```bash
    php artisan serve
    ```
    Проєкт буде доступний за адресою `http://127.0.0.1:8000`.
